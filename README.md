## Featured project

**[Semantic Code Search Engine](https://github.com/Rachel-Mathew25/semantic-code-search)**
> Natural language search over Python codebases — no keyword matching, pure semantic retrieval

- AST-based chunking via Python `ast` module, preserving function and class boundaries
- Two-stage retrieval: vector similarity (ChromaDB) → cross-encoder reranking
- **90% Precision@3** · **13.5ms avg query latency** benchmarked on `psf/requests`
- Precision improved from 63% → 90% after filtering test files from index — a real debugging decision, not a tuning trick

`Python` `FastAPI` `ChromaDB` `sentence-transformers` `Streamlit`
